# freakybrew
Homebrew beer Temperature Control

What you will need:
 - Raspberry Pi 
 - BMP180/183 sensor
 - Relay
 - Wifi dongle
 - Cloud server Ubuntu 2 CPU and 2 Gb Ram
 - Brew Belt
 
Get Started:
 - Download Client folder into your Raspberry Pi
 - Download Server folder into your cloud server
